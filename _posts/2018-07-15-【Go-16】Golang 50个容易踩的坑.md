# 一. 概述
Go 是一门非常简单并且有趣的高级程序开发语言，跟其它语言类似它也有一些`陷阱`。很多陷阱并不是Go本身自己的问题，有一些是因为从其它语言的带来的一些习惯，有些是没有考虑清楚细节导致的。

# 二. 陷阱
1. `左括号不能放在单行`: 很多语言我们可以选择左括号放在哪个位置，但是Go不行，必须放在当前行的最后一个位置
   ```
   // function
   func main() {
      ...
   }
   
   // for
   for i < 50 {
      ...
   }
   
   ...
   ```
